# Pathways Minimal App

This folder contains a minimal Vite + React scaffold to use as a fresh starting point. Drop in your components or replace files in `src/`.

Quick start:

```bash
cd frontend/minimal-app
npm install
npm run dev
```

Dev server will run on `http://localhost:5174` by default.
